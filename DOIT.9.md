## Keyboard / standalone (PWA)

- [x] [S] **`Cmd+,` -> Preferences is BEST-EFFORT in a browser tab; only GUARANTEED in a standalone PWA.** In a normal Chrome tab `Cmd+,` is the browser's Settings accelerator; unlike the hard-reserved combos (`Cmd+T/W/N/Q/Shift+N`), Chrome currently delivers `Cmd+,` to the page, so `event.preventDefault()` suppresses `chrome://settings` and YOLOmux can claim it — BUT this is browser/version-dependent and NOT guaranteed (Safari/Firefox differ; a future Chrome could reserve it). RELIABLE ownership requires an installed PWA (`display: standalone`) or a desktop wrapper, where there is no browser Settings menu. ACTION: keep `Cmd+,` -> Preferences as best-effort (`preventDefault`), but NEVER make it the only path — Preferences must also stay reachable via the Settings menu + the command palette (both exist) so it degrades gracefully. Ties to TODO P7 "installable PWA": shipping a PWA is what makes `Cmd+,` (and other browser-reserved combos) fully YOLOmux's.
